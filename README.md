# Chief Mikey x Secret Souls

A browser-based 2D adventure game built with the Kaboom game engine. The game features multiple scenes, character interactions, sprite-based levels, and sound effects, served via a Koa static file server.

## Tech Stack

- TypeScript
- Kaboom (2D game engine)
- Koa (static file server)
- Webpack (bundler)

## Development

Install dependencies:

```
npm install
```

Build and watch for changes:

```
npm run build:dev
```

Start the development server:

```
npm run start:dev
```

## Production

Build for production:

```
npm run build:prod
```

Start the production server:

```
npm run start:prod
```

The server runs on port 3005 and serves the compiled game from the `docs/` directory.
